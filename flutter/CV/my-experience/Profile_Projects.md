# Profile: Projects

## PROJECT 11 (Flagship): Ayyam App — Islamic Events Calendar & Reminder System
- **Tech Stack:** Flutter, Dart, Bloc (Cubit), Hive, Dio, GoRouter, flutter_local_notifications, Equatable.
- **Architecture:** Feature-Based Clean Architecture (Data, Logic, Presentation layers).
- **Key Achievements:**
  - Designed multi-layered data architecture using Dio (remote), Hive (local cache), and Dart (local calculation) for offline-first operation.
  - Implemented a Repository Pattern orchestrating a cache-first data fetching strategy with automatic fallback.
  - Engineered notification scheduling with timezone-awareness and declarative routing via GoRouter with guards and deep linking.
  - Fully internationalized (i18n) and implemented dynamic theme customization.
